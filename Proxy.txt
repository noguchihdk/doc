■プロキシ設定
netsh winhttp set advproxy setting-scope=machine settings-file=settings.json

■確認
netsh winhttp set advproxy

■もとに戻す (User)
netsh winhttp set advproxy setting-scope=user settings="{\"Proxy\":\"\",\"ProxyBypass\":\"\",\"AutoconfigUrl\":\"\",\"AutoDetect\":true}"

■もとに戻す (machine)
netsh winhttp set advproxy setting-scope=machine settings="{\"Proxy\":\"\",\"ProxyBypass\":\"\",\"AutoconfigUrl\":\"\",\"AutoDetect\":true}"
